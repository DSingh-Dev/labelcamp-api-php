# LabelCamp API PHP
A PHP wrapper for LabelCamp's API.

## Requirements
* PHP 8.1 or later.

## Installation
Install it using [Composer](https://getcomposer.org/):

```sh
composer require bhanwarpsrathore/labelcamp-api-php
```
